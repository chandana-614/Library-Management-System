# Library Management System

A web-based Library Management System developed using **Python, Flask, SQLite, HTML, CSS, Bootstrap, and JavaScript**.

## Features

- User Login (Librarian, Student, Faculty)
- Book Management
- Add Books
- Delete Books
- Issue Books
- Return Books
- Automatic Fine Calculation
- Library Reports Dashboard
- SQLite Database

## Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Installation

Install Flask:

```bash
pip install flask
```

Run the project:

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```
